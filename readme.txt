== Description ==

**Advanced Rating Block** is a custom Gutenberg Block developed with **Gutenberg Native Components**. It allows you to show a rating system in your Gutenberg Editor. It has come with a lot of customization options so that you can easily customize your rating style and make your own design. You can show different types of rating system like Start rating, Heart Rating, Circle Rating and so on.

### Overview
[youtube https://www.youtube.com/watch?v=5rKn2-Bxguc]

### Key Features
* Simple, light-weight, and super-fast
* Built with **Gutenberg Native Components**
* 3 types of Rating system, e.g star, heart & circle rating
* Rating text & iconic label 
* Highly customizable with a lot of customization options

== Installation ==

This section describes how to install the plugin and get it working.
e.g.
1. Upload the plugin files to the `/wp-content/plugins/advanced-rating-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Simply go to Gutenberg Block Editor and you will find a new category with a new Block named "Advanced Rating Block"
4. Drag and drop to use it.
